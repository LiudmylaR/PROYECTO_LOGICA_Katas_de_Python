:ledger: Descripción del Proyecto\
Este proyecto consiste en aplicar los conocimientos adquiridos en el módulo de Python mediante la resolución de diferentes katas, con el objetivo de reforzar la lógica de programación. Las katas se resuelven de distintas formas, fomentando el razonamiento lógico y la búsqueda de soluciones alternativas. En el desarrollo del proyecto se ha intentado no solo que el código funcione correctamente, sino también que esté bien estructurado y documentado, demostrando un entendimiento real de la lógica utilizada. Se han añadido comentarios explicativos, especialmente en aquellas partes del código que han supuesto mayor dificultad.

:pushpin: Objetivos del Proyecto
* Manejar correctamente tipos de datos y funciones incorporadas.
* Utilizar correctamente las estructuras de datos de Python (listas, tuplas, diccionarios, conjuntos) y sus métodos.
* Implementar estructuras condicionales para la toma de decisiones.
* Trabajar con estructuras de iteración como bucles for y while.
* Definir y usar funciones de forma adecuada.
*  y aplicar los conceptos de programación orientada a objetos, mediante la creación y uso de clases.
* Utilizar módulos para organizar y reutilizar código.
* Aplicar buenas prácticas y documentar el código con comentarios explicativos.

:white_check_mark: Resultados y Conclusiones\
A lo largo del desarrollo del proyecto se han resuelto las distintas katas propuestas, aplicando los conceptos fundamentales del lenguaje Python. Como resultado, se realizó un intento de demostrar el manejo de la lógica de programación, el uso de estructuras de datos, condicionales, bucles, funciones y programación orientada a objetos. Además, se ha puesto especial atención en la organización y documentación del código, incorporando comentarios explicativos en las partes más complejas. En conclusión, este proyecto ha permitido consolidar los conocimientos adquiridos, mejorar la capacidad de análisis y reforzar buenas prácticas de programación, consolidando una base para futuros proyectos de análisis y gestión de datos.
